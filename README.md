# Conway's Game of Life in modern C++ with SFML

## How to run it
Install SFML on Ubuntu:
```
sudo apt-get install libsfml-dev
```

Compile the source
```
g++ *.cpp -o exec -lsfml-graphics -lsfml-window -lsfml-system
```

Run it
```
./exec
```
